# ✅ SEO Meta Tag Checklist

## 1. Essential SEO Tags

| ✔   | Tag                         | Example                                                             |
| --- | --------------------------- | ------------------------------------------------------------------- |
| ☐   | `<title>`                   | `<title>Home - My Brand</title>`                                    |
| ☐   | `<meta name="description">` | `<meta name="description" content="Your compelling page summary.">` |
| ☐   | `<link rel="canonical">`    | `<link rel="canonical" href="https://example.com/page">`            |

---

## 2. Technical Tags

| ✔   | Tag                                   | Example                                                                  |
| --- | ------------------------------------- | ------------------------------------------------------------------------ |
| ☐   | `<meta name="robots">`                | `<meta name="robots" content="index, follow">`                           |
| ☐   | `<meta charset="UTF-8">`              | `<meta charset="UTF-8">`                                                 |
| ☐   | `<meta name="viewport">`              | `<meta name="viewport" content="width=device-width, initial-scale=1.0">` |
| ☐   | `<meta http-equiv="X-UA-Compatible">` | `<meta http-equiv="X-UA-Compatible" content="IE=edge">`                  |
| ☐   | `<meta name="referrer">`              | `<meta name="referrer" content="strict-origin-when-cross-origin">`       |

---

## 3. Branding & Author

| ✔   | Tag                    | Example                                             |
| --- | ---------------------- | --------------------------------------------------- |
| ☐   | `<meta name="author">` | `<meta name="author" content="Your Name or Brand">` |

---

## 4. Open Graph (Facebook, LinkedIn, WhatsApp, etc.)

| ✔   | Tag              | Example                                                                 |
| --- | ---------------- | ----------------------------------------------------------------------- |
| ☐   | `og:title`       | `<meta property="og:title" content="Page Title">`                       |
| ☐   | `og:description` | `<meta property="og:description" content="Brief summary of the page.">` |
| ☐   | `og:type`        | `<meta property="og:type" content="website">`                           |
| ☐   | `og:url`         | `<meta property="og:url" content="https://example.com/page">`           |
| ☐   | `og:image`       | `<meta property="og:image" content="https://example.com/img.jpg">`      |
| ☐   | `og:site_name`   | `<meta property="og:site_name" content="My Brand">`                     |

---

## 5. Twitter Cards

| ✔   | Tag                   | Example                                                             |
| --- | --------------------- | ------------------------------------------------------------------- |
| ☐   | `twitter:card`        | `<meta name="twitter:card" content="summary_large_image">`          |
| ☐   | `twitter:title`       | `<meta name="twitter:title" content="Page Title">`                  |
| ☐   | `twitter:description` | `<meta name="twitter:description" content="Brief summary.">`        |
| ☐   | `twitter:image`       | `<meta name="twitter:image" content="https://example.com/img.jpg">` |
| ☐   | `twitter:site`        | `<meta name="twitter:site" content="@yourhandle">`                  |

---

## 6. Favicon & Icons

| ✔   | Tag                             | Example                                                      |
| --- | ------------------------------- | ------------------------------------------------------------ |
| ☐   | `<link rel="icon">`             | `<link rel="icon" href="/favicon.ico" type="image/x-icon">`  |
| ☐   | `<link rel="apple-touch-icon">` | `<link rel="apple-touch-icon" href="/apple-touch-icon.png">` |
| ☐   | `<link rel="manifest">`         | `<link rel="manifest" href="/site.webmanifest">`             |

---

## 7. Language & Locale

| ✔   | Tag                 | Example                                                              |
| --- | ------------------- | -------------------------------------------------------------------- |
| ☐   | `<html lang="en">`  | Add to opening `<html>` tag                                          |
| ☐   | Hreflang (optional) | `<link rel="alternate" hreflang="id" href="https://example.com/id">` |

---

## 8. Schema Markup (Structured Data)

| ✔   | Tag            | Example                                             |
| --- | -------------- | --------------------------------------------------- |
| ☐   | JSON-LD schema | `<script type="application/ld+json">{...}</script>` |

Use appropriate type: Organization, WebPage, Article, LocalBusiness, etc.

---

## 9. Optional / Legacy

| ✔   | Tag                                   | Example                                               |
| --- | ------------------------------------- | ----------------------------------------------------- |
| ☐   | `<meta name="keywords">` (deprecated) | `<meta name="keywords" content="keyword1, keyword2">` |
